Calculator Hints

Although it is vital to be able to generate solutions to questions using correct methods, it is also very handy to be able to check our answers using a calculator.

This page lists the common functions of a CASIO calculator that can help us perform these checks.

Mode Check

Check that your calculator is in the correct mode.

sin(30)

If you do not get sin(30) = 0.5, you are in the incorrect mode.

To change to DEGREES

[shift] [setup] 3
