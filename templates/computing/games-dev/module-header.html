Game Engine for 2019-2020

In 2018-19, the club used javascript and the p5 libraries to create its game projects. However, as the department will be focussing on python for all programming, we have decided to use the excellent GODOT engine for 2019-2020.

Godot meets a number of criteria, such as:

  • Multi platform and be used on Windows and Mac computers.
  • Is easily installed, and can be made to run at home with minumim expereince.
  • It is open source and free to use, again encouraging pupils to install the software at home.
  • It is well documented with a host of video tutorials.

While GODOT is not exactly python based, its scripting language GScript is based on python, and the two are very similar. GODOT is often described as a lighter contender to Unity and Unreal game engines and is certainly capable of producing very professional games quickly.

I'm really looking forward to seeing what the Games Club makes in the new year with this exciting software package.
