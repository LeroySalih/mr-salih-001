What is Computer Modelling?

Definition:

A computer-based model is a computer program that is designed to simulate what might or what did happen in a situation.

Software Modelling allows us to consider the interaction of real world objects.

Wikipedia, 2018

Specifying Dimensions

  • Click to start the rectangle (do not drag!)
  • Type the dimensions (e.g. 2m,1m)
  • Press Enter

Using the Tape Measure

You can use the tape measure to generate guide lines that help you plan the placement of shapes.
  • Click to start tap emeasure
  • Move the mouse in the desired direction
  • Enter distance (e.g. 0.1m)
  • Press enter

Pencil Lines

Pencil Lines allow us to add Vetrices and Edges to existing shapes.

Pencil Lines also allow us to create new faces.

Polygons

Polygons allow us to create regular polygons.

You can specify the number of sides before selecting the center point.

Merging Shapes

Drawing shapes over existing shapes will merge the verticies to form a new shape. This new set of verticies can either be treated seperately, or can be merged in to a single new shape using the eraser tool.

What are Components?
To be able to:
  • Crate a simple component.
  • Edit a component.

Components are

  • Components are a way to create reusable objects.
  • Components can be shared between files and people.
  • Components allow us to produce repeated shapes, very quickly.
  • Components allow us to move shapes, without them becoming sticky.
