What is Layout?

An Element is a discrete unit of design within the webpage.
A Component is a collection of elements with a common purpose.
A Section is a related collection of components.
Layout describes the placement of these sections within the webpage.

Good Layout Guides

  • Users are pleased by the design, but drawn to the content.
  • Users can move easily via intuitive navigation.
  • Users can recognise that each page belongs to the site.
  • Principles of Beautiful Web Design, BEAIRD & GEORGE, 2014
