Project - Bank System
In this module we are going to practice Creating tables to hold our data.

Project

To be able to practice creating and using database tools. In this project, you will:

  • Create the tables required for a business system.
  • Create the updates required to keep the system correct.
  • Create the UI to display .

To be able to design a table, choosing appropriate data types.

To be able to build a table, using MYSQL commands.

Requirements

Using a python based environment, create a Bank Account system.

    Create a Customer table, that contains:
    • CustomerID
    • firstName
    • familyName
    • currentBalance
    • Is active
  1. Create a transactions table, that contains:
    • TransactionId
    • CustomerId
    • Date
    • Type (deposit or withdrawl)
    • Amount
  2. Your system must allow the following actions:
    • Create Customer
    • Make Customer Active/Inactive
    • Deposit money
    • Withdraw Money, preventing the user from withdrawing more money than available.
    • List All transactions
    • List All transactions of a type
    • List all transactions since a date
  3. Extension: As an extension, your system will:
    • Use transactions to ensure that tables are kept consistent
    • Use a UI other than a command prompt (TkInter, etc)
